# Dotfiles

## About
These are my dotfoles that I use on my desktop.

## Requirements

### Arch Linux

Required software:
- Hyprland (with Hyprlock, Hypridle)
- Hyprshot
- swww
- Rofi
- waybar
- SwayNC
- SwayOSD
- kitty
- Nemo
- Overskride, (flatpak version)
- brightnessctl

Also, I used [Gruvbox GTK Theme](https://github.com/Fausto-Korpsvart/Gruvbox-GTK-Theme) in my configs.
